# 数组

## 数组基础

存放在连续逻辑内存空间上相同类型数据的集合称为数组。
+ 逻辑空间连续
+ 狭义数组元素不可删除，只能替换

> [!info]
> 在虚拟存储空间上，多维数组存储是否连续取决于对应的编程语言和编译器，如支持用户操作内存地址的 C 语言直接分配连续的虚拟存储空间，而不支持直接操作内存地址的 Java 则不暴露对应指针

## 常用算法

### 二分查找

前提条件：
+ 数组有序
+ 无重复元素

确定好循环不变量，如使用左闭右闭区间，则循环不变量应当根据区间定义操作，此时left\==right 有意义，因此循环不变量带等号

```cpp
while(left<=right){
	mid=(left+right)/2;
	if(nums[mid]==target) return mid;
	else if(nums[mid]>target){
		right=mid-1;
	}
	else{
		left=mid+1;
	}
}
return -1;
```

如果采用左闭右开区间

```cpp
while(left<right){
	mid=(left+right-1)/2
	if(nums[mid]==target) return mid;
	else if(nums[mid]>target){
		right=mid;
	}
	else{
		left=mid+1;
	}
}
return -1;
```
