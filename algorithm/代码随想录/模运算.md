# 模运算

某些题目，由于要计算的答案非常大（超出 64 位整数的范围），会要求把答案对 $10^9+7$ 取模。如果没有处理得当的话，会 WA（错误）或者 TLE（超时）。

> [!tips] 
> 这是一个质数，因此天然支持模除运算
## 加减法与乘法的模运算

以下是两个恒等式：
$$
  (a+b)modm=((amodm)+(bmodm))modm
$$
$$
(a⋅b)modm=((amodm)⋅(bmodm))modm  
$$
但这两个恒等式无法保证中间计算结果一定不会发生溢出，且出现负数乘法时也不可以直接使用，在真实的代码实现中，通常使用以下方式：

```cpp
// 加
(a + b) % MOD

// 减，b 在 [0,MOD-1] 中，通过 +MOD 保证结果非负
(a - b + MOD) % MOD

// 把任意整数 a 取模到 [0,MOD-1] 中，无论 a 是正是负
(a % MOD + MOD) % MOD

// 乘（注意使用 64 位整数）
a * b % MOD
```
