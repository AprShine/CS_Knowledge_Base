# Open Ondemand

## 什么是 Open Ondemand

Open OnDemand (OOD) 是由俄亥俄超级计算机中心开发的一个开源 Web 平台，旨在让用户仅通过浏览器就能轻松访问高性能计算 (HPC) 资源。它的核心优势在于无需用户安装任何客户端或掌握复杂的 Linux 命令行知识，即可直接在网页端完成文件管理、作业提交与监控等操作。此外，它还集成了 Jupyter Notebook、RStudio 以及远程桌面等图形化交互应用，极大地降低了超算的使用门槛，帮助科研人员和学生更专注于研究本身，目前已被全球众多高校和科研机构广泛采用，特点如下：
+ **零门槛访问**：用户无需安装客户端或掌握复杂的 Linux 命令行，即可直接通过网页登录超算系统。    
- **功能全面**：集成了文件管理、作业提交与监控、以及图形化交互应用（如 Jupyter Notebook、RStudio 或远程桌面）。
- **降低难度**：极大地简化了科研人员和学生使用 HPC 的流程，使计算资源更易于获取。
