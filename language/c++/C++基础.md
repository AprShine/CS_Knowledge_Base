## 静态局部变量，全局变量，局部变量的特点，以及使用场景？

- **静态局部变量**：函数**内**定义，用static修饰，**生命周期为整个程序**，该变量无法在作用域外修改，**只能被初始化一次**，之后每次调用都保持上以上调用结束时的值，存储在数据段。
- **全局变量**：函数**外**定义，生命周期是**整个程序运行区间**，程序中可以在任何地方访问，创建在数据区。
- **局部变量**：函数**内**定义，**作用域和生命周期只能在声明该变量的函数体内或者类内访问**，每次调用重新创建，生命周期随着函数的返回而结束，创建在**栈**上。

> [!info]
> 1. 局部变量的值不确定，静态局部变量和全局变量默认初始化为0；
> 2. 局部变量存储在堆栈段，静态局部变量和全局变量存储于数据段/BSS段
> 	+ **BSS段**：符号起始块段，特点是不占用磁盘空间，仅记录段大小，并在程序加载时由 OS 自动清零，适用于未进行自定义初始化的静态局部变量/全局变量

### 怎么用

+ 全局变量适用于存储**很多地方都要用**的数据，但不应当泛滥，使用时写好注释；
+ 静态局部变量适用于在函数内执行需要记忆上次一函数执行时的某种状态的情形，如计数器等；
+ 一般无特殊情况，使用局部变量

## 指针和引用的区别

+ 指针
以变量的形式定义，存储另一个变量的内存地址，可以使用引用访问目标变量，可以重新赋值以指向其他对象，可以为空，可以是常量 const，不为常量时支持指针运算，如移动地址（合法）。
+ 引用
是变量的**别名**，绑定后**不能修改**，不能为空，使用时无需解除，不可声明为常量，绑定后与目标共享内存，无独立存储空间，生命周期与绑定对象一致，无需手动管理内存。

举一个例子：

```cpp
int i=10;
# 指针
int *index_i=&i;
# 引用
int &j=i;
```

### 怎么用

+ 指针适用于遇到可能需要处理 NULL 的情况（可选参数）/ 需要改变指向对象（遍历链表）/需要动态内存分配（new/delete），遇到多级间接访问时
+ 引用通常用于函数参数传递 / 绑定有效对象以实现链式调用 / 运算符重载等场景
